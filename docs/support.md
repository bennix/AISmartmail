# myMail Support Page Draft

Use this draft for the public support URL required by App Store Connect. Publish it before submission.

## Getting Help

For support, include:

- macOS version.
- myMail app version.
- Mail provider or server type.
- Whether the issue affects receiving, sending, attachments, OAuth, or AI indexing.
- Any visible error message.

Do not send passwords, app-specific passwords, OAuth tokens, or API keys.

## Common Setup Notes

myMail supports generic IMAP, SMTP, and POP3 server settings. SSL/TLS or STARTTLS is recommended whenever the server supports it. Older unencrypted ports are available only for custom servers that require them.

AI features are optional. Mail receiving, reading, searching, and sending work without an AI API key. Vector indexing is used for AI question answering and attachment-content search; it can be skipped.
